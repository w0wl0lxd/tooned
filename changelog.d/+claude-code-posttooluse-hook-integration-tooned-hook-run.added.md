Claude Code `PostToolUse` hook integration: `tooned hook run --claude-code` reads the `tool_output` stdin field, replaces it in place via `hookSpecificOutput.updatedToolOutput` when a smaller TOON encoding exists, and always exits 0 (never blocks a tool call). `tooned hook install --claude-code [--scope user|project] [--mcp]` verifies the `tooned` binary resolves on `PATH` first, then idempotently merges a `PostToolUse` entry (matcher `Bash|Read|Grep|WebFetch|^mcp__`) into `settings.json`, leaving any pre-existing foreign hook entry byte-for-byte untouched. `tooned hook uninstall --claude-code` removes only tooned's own entry.
