**tooned-cli:** added plugin-wrapped agent hooks for OpenCode, Kilo Code, and Pi. `tooned hook install --opencode|--kilo|--pi` writes a TypeScript plugin/extension file that calls `tooned hook run --<flag>` with a Claude-compatible `tool_output` payload, then mutates the agent's tool result only when TOON is smaller. `uninstall`, `status`, and `doctor` are supported for all three agents.
