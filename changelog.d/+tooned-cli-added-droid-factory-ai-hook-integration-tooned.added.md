**tooned-cli:** added Droid (Factory AI) hook integration. `tooned hook run --droid` uses Droid's `hooks.PostToolUse` JSON format (`toolName` / `toolInput` / `toolOutput`). Droid only supports `additionalContext` in `PostToolUse`, which would keep the original JSON in context, so the hook passes through and prints nothing; use `tooned wrap -- <cmd>` or `... | tooned pipe` for TOON-only output with Droid. `tooned hook install --droid` writes `.factory/hooks.json` (project) or `~/.factory/hooks.json` (user), `uninstall --droid` only removes tooned's own entries, and `status`/`doctor` report Droid installations.
